Se debe crear una carpeta en disco local C: con el nombre de Store Managenet Software, guardar aquí todos los archivos; dentro de esta carpeta SMS una nueva que se llame Database (aqui incluir el archivo store.db). Concluido esto, se puede correr interfaz 2.1 :)

El programa esta realizado con sqlite, para poder correrlo es neesario tener instalado SQ en el ordenador de puede descargar de la página http://sqlitebrowser.org/
